# CLAUDE.md — 09_統計表-game

第九單元「統計表」互動工具，包含讀一維表格、讀二維表格與製作統計表。

## 技術框架

- 純 HTML + CSS + 原生 JavaScript
- 無 build 流程
- GitHub Pages 靜態部署

## 主要頁面

- `index.html`：導覽首頁
- `table1d.html`：讀一維表格
- `table2d.html`：讀二維表格
- `make-table.html`：製作統計表

## 不能動的地方

- 題目卷 DOCX 與 `u9_text.txt` 是參考資料，不要任意刪除。
- `.bak` 是修改前備份。

## 部署後驗證清單

- 開啟 GitHub Pages 首頁。
- `table1d.html` 可完成查值、填空、反推。
- `table2d.html` 可讀統計表/時刻表/票價表情境。
- `make-table.html` 可完成計數、合計與應用題。
